# ❤️ Cardio Clever – Heart Disease Prediction System
Cardio Clever is a web-based application that predicts the risk of heart disease based on user-provided medical and lifestyle parameters using data-driven analysis.

## Features
* Predicts heart disease probability
* Interactive sliders for user input
* Real-time risk percentage visualization
* User-friendly and responsive web interface
* Preventive health awareness platform

## Technologies Used
* HTML
* CSS
* JavaScript
* D3.js

## How to Run
1. Clone the repository
2. Open the project folder in **VS Code**
3. Install **Live Server** extension
4. Right-click `index.html` → **Open with Live Server**
5. Enter health parameters to view prediction results

## Input Parameters
* Sex
* Resting Blood Pressure
* Maximum Heart Rate
* Chest Pain Type
* Number of Major Vessels
* ST Segment Slope
* Thalium Test Result
* Alcohol Consumption
* Weight
* Family History
* Smoking Habit
* Left Hand Pain

## Output
* Displays heart disease risk percentage
* Dynamic graphical visualization
* Instant updates based on input changes

## Note

This project is developed for **academic and learning purposes** using statistical and data-driven logic for prediction.
